This is not affiliated with any institution. It's the work of someone who finds this field genuinely fascinating and believes that **understanding financial crime is the first step to preventing it**.

## Methodology

All reports follow a consistent investigative workflow:

1. Identify seed address from public sources
2. Trace all transactions via mempool.space and Blockchair
3. Classify address types and detect clustering patterns
4. Identify timing anomalies, fee anomalies, and structural patterns
5. Map findings to ACAMS CAMS typology framework
6. Cross-reference with FATF VA Guidance and Basel AML Index
7. Apply jurisdictional analysis
8. Identify residual risk and VASP obligations

**Tools used:** Arkham Intelligence · mempool.space · Blockchair · OXT · OSINT public sources
